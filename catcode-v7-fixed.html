📖 Help
📝 Text Mode — v1 Cat Translator
Reversible UTF-8 → compact Cat64 shell. No password needed.
📝 Text Mode — v2 CatCipher
Real PBKDF2 + AES-GCM encryption. Only the correct password can decrypt.
📁 File Mode — CatLock
Encrypt any file (max 100 MB) with a cat password. Creates a .catlock file that can only be opened by decrypting it back. Uses PBKDF2 + AES-256-GCM. Original filename and type are preserved inside.

After encoding text, click play to hear your message as a cat melody! Each message sounds different.
